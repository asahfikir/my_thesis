Self-Attention

The core mechanism of understanding context

Attention allows the model to weigh the importance of different words when processing each token. It answers: "Which other words should I pay attention to?"

Example Sentence

"The cat sat on the mat because it was tired"

Query (Q)

"What am I looking for?"

Represented by vector Q

Key (K)

"What do I contain?"

Represented by vector K

Value (V)

"What information do I pass?"

Represented by vector V

Stacked Layers

Building depth through repetition

LLMs stack these blocks 12-96 times (GPT-3 has 96 layers). Each layer refines the representation, building increasingly abstract features.

Layer 1
Surface patterns
Layer 12
Syntax & grammar
Layer 24
Semantic meaning
Layer 48
Context & reasoning
Layer 96
Abstract concepts

Residual Connections & Layer Norm

Each sub-layer has a residual connection (skip connection) followed by layer normalization. This helps gradients flow during training and stabilizes learning.

8

Output Generation

Predicting the next token

Finally, the model outputs probability distributions over the entire vocabulary. It predicts the most likely next token, then feeds it back as input to generate the next (autoregressive generation).
